# Verdure.UniversalController
讲述一个万能控制器的复刻过程，以及代码的整理。

## 项目介绍

本项目是讲述复刻一个B站up主黑人黑科技制作的万能的遥控器的过程，主要包括硬件的焊接，以及实物展示，和代码的整理，由于作者直接打包的代码，所以本人经过整理将代码整理到了GitHub，如果有侵权的问题，那可以通知本人，本人可以及时的删掉。
